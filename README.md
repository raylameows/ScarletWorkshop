# Scarlet Workshop

Scarlet Workshop is a collection of tools for [Scarlet](https://www.roblox.com/games/125826368944762/Scarlet) custom map creation.

Custom maps are created in Roblox Studio and tested using private servers. Scarlet Workshop allows you to serialize your map into a string and load it directly in-game, allowing for simple testing and easy sharing.

This collection of tools consists of:

### Serializer

The Serializer converts your map into a long string which can then be loaded through Scarlet's in-game custom map system in a private server. This allows you to test changes in real time.

Serialized map codes may also be read back by pasting them into the Serializer's **Readback** field, allowing you to view or edit an existing map.

### Map Selector

Managing multiple maps can quickly become inconvenient when each one requires its own place.

The Map Selector allows you to seamlessly switch between the maps you're currently editing, making it easier to work on multiple projects without having to constantly switch places.

### Lighting Port

This tool allows you to import and export a map's lighting settings.

Exported lighting can later be imported again, allowing you to preview and edit it.

## Gists

Custom map codes may be uploaded as [GitHub Gists](https://gist.github.com/) and then loaded using the link, as the game will fetch the code directly from the Gist. This allows for easier sharing and better management of map codes.

## Community

If you require assistance with using the tools, you may ask for help in our [Discord community server](https://discord.gg/x9nXjbkxan).

You may also submit your custom map for consideration to be officially added to Scarlet. Maps selected for inclusion may appear in public servers as part of the normal map rotation.

Please note that submission does not guarantee inclusion, and maps may be reviewed for quality, gameplay, performance, and overall fit within the game.
